# Custom Theme

Use `theme` prop on `<Notifin />` to override visuals.

```tsx
import { Notifin } from "@khencahyo13/notifin-react";

<Notifin
  theme={{
    dialogToneClasses: {
      success: "border-emerald-300 bg-emerald-50 text-emerald-950",
      error: "border-rose-300 bg-rose-50 text-rose-950",
    },
    iconToneClasses: {
      success: "border-emerald-400 bg-emerald-100 text-emerald-700",
      error: "border-rose-400 bg-rose-100 text-rose-700",
    },
  }}
/>;
```

### `dialogToneClasses`

Container tone classes per type (`default`, `success`, `error`, `warning`, `info`, `loading`).

### `iconToneClasses`

Icon wrapper tone classes per type.

### `icons`

Custom icon components per type. See [Custom Icons](./custom-icons).

### `schemes`

Per-color-scheme override:

- `schemes.light`
- `schemes.dark`

```tsx
<Notifin
  colorScheme="system"
  theme={{
    schemes: {
      dark: {
        className: "ring-1 ring-white/10",
      },
    },
  }}
/>
```
