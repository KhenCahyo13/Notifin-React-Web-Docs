# Notifin

`@khencahyo13/notifin-react` is a function-first alert dialog library for React, built on top of Radix Alert Dialog.

It is designed for high-attention confirmations and action feedback with a simple calling API.

## Quick Start

```tsx
import { Notifin, notifin } from "@khencahyo13/notifin-react";

export default function AppLayout() {
  return (
    <>
      <Notifin />
      <button
        onClick={() =>
          notifin.success("Profile updated", {
            description: "Your changes have been saved.",
          })
        }
      >
        Save
      </button>
    </>
  );
}
```
