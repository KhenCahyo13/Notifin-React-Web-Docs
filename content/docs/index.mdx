# Notifin

`@khencahyo13/notifin` adalah function-first alert dialog library untuk React, dibangun di atas Radix Alert Dialog.

Library ini cocok untuk skenario konfirmasi penting atau feedback action yang butuh fokus pengguna, dengan API pemanggilan yang sederhana.

## Quick Start

```tsx
import { Notifin, notifin } from "@khencahyo13/notifin";

export default function AppLayout() {
  return (
    <>
      <Notifin />
      <button
        onClick={() =>
          notifin.success("Profile updated", {
            description: "Perubahan sudah disimpan.",
          })
        }
      >
        Save
      </button>
    </>
  );
}
```
